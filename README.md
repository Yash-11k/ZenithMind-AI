# 🧠 ZenithMind AI — Student Burnout Prediction System

> Predicting academic burnout through behavioral analytics using Machine Learning.

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://zenithmind-ai-yash11k.streamlit.app/)

---

## 📌 Overview
ZenithMind AI is an end-to-end ML web application that predicts student burnout risk 
based on 15 behavioral, psychological, and academic features. 
Trained on a dataset of 1M+ student records.

---

## 🚀 Live Demo
https://zenithmind-ai-yash11k.streamlit.app/
---

## 🧠 ML Pipeline
- **EDA** — Analyzed 20+ features across 1M rows
- **Models** — Random Forest & XGBoost (Regressor) , Train & Test 71 Accuracy 
- **Key Predictors** — Sleep hours, screen time, exam pressure, anxiety score
- **Output** — Burnout Score (0–3) → Low / Moderate / High Risk

## 📊 Features Used
- Study Hours, Sleep Hours, Screen Time
- Stress Level, Exam Pressure, Anxiety Score
- Depression Score, Social Support, Financial Stress
- Family Expectation, Physical Activity
- Academic Performance, Stress-Sleep Ratio
- Pressure Gap, Academic Overload
